# Components List – LED Blink

| Component | Quantity | Description |
|---------|---------|-------------|
| Arduino Board | 1 | Uno / Nano / Mega |
| LED | 1 | Any color |
| Resistor | 1 | 220Ω – 330Ω |
| Breadboard | 1 | Half-size |
| Jumper Wires | 2–3 | Male-to-male |

---

## ⚠️ Notes
- Always use a resistor with an external LED
- Built-in LED is connected internally to pin 13
- Wrong polarity may prevent LED from glowing
